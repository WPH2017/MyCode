#运算符

##1. 数学运算符
\+ - * / % ()
####最重要的是整数和浮点数的相互转换

##2. 关系运算符
\> < >= <= == != === !==

##3. 逻辑运算符
! && ||
**运算顺序 非 与 或**

**与、或的短路语法**

##4.赋值运算符 特点：可以用逗号隔开连续赋值。
= += -= *= /= %= ++ --
