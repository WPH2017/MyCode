# CSS进阶书写

## 属性简写模板
```
{
    position:relative;/* 定位 */
    
    float:left;/* 浮动 */
    
    margin:;/* 外边距 */
    
    padding:;/* 内边距 */
    
    font:;/* 字体 */
    
    border:;/* 边框 */
    
    background:;/* 背景 */
    
    box-shadow:;/* 阴影 */
    
    animation:;/* 动画 */
    
    /* 特化属性 */
}
```
